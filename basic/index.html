Let's talk about pug

What is pug?

Pug is a template engine implemented in javascript using node. It compiles into standard HTML. Pug relies on indention rather than opening and closing tags to create elements. Pug was formally known as Jade.

Why use pug?

  • Easier to write html
  • Ability to use templates that make code more reusable
  • Code becomes cleaner and easier to read
  • Awesome file icon (if you use file-icons which you should)
